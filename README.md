# HSKD

**HSK** is an implementation of the Handshake Protocol.

## Install

```
$ git clone git@github.com:handshakecompany/hskd.git
$ cd hskd
$ npm install
$ ./bin/hskd
```

See the [Beginner's Guide][guide] for more in-depth installation instructions.
